# Project notes

## Setup

0. Have CMake installed and on $PATH environment variable
1. Create a build directory
2. Change current working dir into build directory
3. run `cmake -G "Visual Studio 17 2022" -A x64 ..` or change the generator for your use-case
4. Open project solution in `./build/Grinder.sln` or other generated file

Alternatively run `generate.bat` and pray it works
